# مطعم دار المذاق

موقع من صفحة واحدة لمطعم عربي، مصمم بواجهة عربية متجاوبة باستخدام HTML وCSS وJavaScript فقط.

## الملفات

- `index.html` — بنية الصفحة والمحتوى.
- `styles.css` — التصميم، الألوان، الحركة، والتجاوب.
- `script.js` — تأثير ظهور العناصر عند التمرير.

## التشغيل

افتح ملف `index.html` في المتصفح مباشرة.
